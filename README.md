
# Swiggy Clone 🌇

Swiggy clone - Swiggy is an Indian online food ordering and delivery platform.

## Tech Stack Used 👩‍💻

- HTML
- CSS
- Javascript
- React
- Redux


## Run Locally 💻

Clone the project

```bash
  git clone https://github.com/YogendraRajpoot/Swiggy-Clone.git
```

Go to the project directory

```bash
  cd my-project
```

Start

```bash
  start ./index.html
```
